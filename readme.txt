・ルーチンタスク
日々のタスクを管理するためのシンプルなタスク管理アプリです。GoogleChromeの拡張機能として作られていますのでそちらで使用してください。

・環境
manifest_version : 2
VueCLI : 2.9.6

・セットアップ
1.ダウンロードしたport-master.zipを解凍してください。
2.Chromeのオプション→その他のツールから拡張機能のページを開いてください。
3.デベロッパーモードをオンにし、”パッケージ化されてない拡張機能を読み込む”を押し、解凍したport-masterフォルダを選択してください。
4.アドレスバーの横にアイコンが表示されればOKです。アイコンをクリックすることでアプリを起動できます。

・使い方
画面下部の入力フォームにタスクの内容を入力し、ENTERキーか追加ボタンで確定すると上部にタスクが追加されます。

追加されたタスクは状態ボタンでステータス(途中/管理)を切り替えられます。

不要なタスクは　”CTRキーを押しながら”　削除ボタンを押すことで削除できます。

また画面最上部のボタンで任意のステータスのタスクだけを表示できます。

残っているタスクは完了状態にしていても午前7時にステータスが”途中”に戻ります。
